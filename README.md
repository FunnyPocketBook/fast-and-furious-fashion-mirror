## Project setup
```
pnpm install
```

### Compiles and hot-reloads for development, can also be used to preview
```
pnpm run serve
```
or
```
pnpm run dev
```

After executing this command, check the terminal for a localhost url to preview the page.

### Compiles and minifies for production
```
pnpm run build
```
